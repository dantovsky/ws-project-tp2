Esta aplicação corresponde ao projeto de Web Semântica do Mestrado em Engenharia Informática. O conceito principal deste trabalho é sobre a manipulação e uso do SPARQL e a frameword Django.

A fase inicial deu-se pela busca de uma fonte de dados para popular um repositório no GraphDB. O grupo utilizou um conjunto de dados no formato CSV e utilizou a ferramenta Tabular(OntoRefine) presente no GraphDB, destinada à importação de dados a partir de vários tipos de formatos, permitindo assim, a transformação e refinação dos dados antes de popular o repositório do GraphDB.
